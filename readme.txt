Put a folder named /spring here, and extract a portable install of Spring engine to it
Install the map Darkside Remake (http://zero-k.info/Maps/Detail/7291)
    and the Spring Tutorial Game (http://springfiles.com/spring/other/springtutorialgame-sdz-download)
